# bull — 今日の献立表 (Today's Menu)

A simple web page that displays today's meal plan (献立表) with meal times, dish names, descriptions, and calorie counts.

## Usage

Open `index.html` in your browser — no build step or server required.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main page structure |
| `style.css` | Styling |
| `menu.js` | Menu data and rendering logic |

## Customising the Menu

Edit the `MENU_DATA` array inside `menu.js` to change dishes, descriptions, or calorie values.
